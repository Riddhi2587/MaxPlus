To initialize a matrix 

A = Matrix( [ [ … ], 
		      [ … ]])
Example:
a= Matrix([[1, 2, 3], [4, 5, 6], [7, 8, 9]])


Functions:

1. Add
   If a and b are 2 matrices, 
   c = add(a, b)

2. Multiply
   If a and b are 2 matrices, 
   c = multiply(a, b)
   To view c,
   print(c)

   If a is a matrix and b is a scalar
   c = multiply(b, a)

3. Power
   c = power(a, n)

4. Exponent
    If a is a matrix
    c = exp(a)

    If a is a scalar
    c = exp(a)
    

5. Gamma
    c = a.gamma()

6. Permanent
    c = maper(a)
    print(c)

7. Eigenvalue
    c = a.eigenval()

8. Eigenvec
    c = a.eigenvec()

9. Period
    c = period(a)

10. Characteristic equation
    c = char(a)

11. Period 
    c = period(a)
